About EndCrypt

EndCrypt uses advanced chaotic map algorithms to encrypt images, providing secure visual cryptography. The main techniques used are:

  • Arnold Cat Map: A spatial transform that shuffles the pixels of an image in a chaotic yet reversible manner.
  • Logistic Map: A non-linear chaotic system used to generate pseudo-random sequences for encryption.
  • Sine Map: Another chaotic map used to enhance the randomness of the encryption process.

Multiple encryption levels apply different iterations and complexity to the encryption process, allowing you to choose the balance between security and performance.
